# 项目概览

我们已经开始接受 Anytype 桌面版的翻译。

该项目通过 [Crowdin 平台](https://crowdin.com/project/anytype-desktop)进行管理。

您还可以直接将您的翻译贡献到 [`l10n-anytype-ts`](https://github.com/anyproto/l10n-anytype-ts) 存储库。

## 支持的语言
目前，应用程序不支持从右到左的语言，因此我们没有提供这些语言的翻译。

如果您的语言尚未在 Crowdin 上列出，请在 [贡献者社区](https://github.com/orgs/anyproto/discussions/45) 留下评论，我们会尽快将其添加。

## 下一步
1. 启动 Anytype 手机版的本地化
2. 优化应用程序以支持从右到左的语言，并提供相应翻译。
